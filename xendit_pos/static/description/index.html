Use Xendit to accept POS payments in Odoo. With Xendit, there is no need for special hardware.

  • Accept all Xendit payment methods, including eWallets, Direct Debit, Credit Cards, and more!
  • No need for special POS equipment, get started with a basic laptop or tablet
  • Free download and support for the Odoo <> Xendit integration

Installation overview

Please see our documentation for the most up to date information

  1. Download the Xendit POS add-on from the Odoo Marketplace.
  2. Unzip module file to the Odoo add-ons folder
  3. Re-start Odoo
  4. Turn on the developer mode > Update the apps list
  5. Find the Xendit app and install it
  6. Enter your Xendit API keys, and start accepting payments!

Requirements

  • Odoo version 14 and above
  • Odoo.sh or On premise, Odoo Online not supported
  • Laptop or Tablet for POS use
